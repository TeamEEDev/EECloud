EECloud makes sure it does not waste your system resources. Less than 50MB of RAM is required to run EECloud. It also consumes the least processing power possible to make sure your plugins run really smooth.

Want to install a plugin? Simply drag it into the Plugins folder. There are no configuration files or any other software you will need. Want to remove one? Remove it from your Plugins folder and it's already uninstalled!

Run it on desktop or the Cloud with no changes in your code. Install it on your PC, or instantly run from your USB drive. Write plugins in C#, VB.NET, F#, Javascript .NET, or any other language. EECloud has no limitations!
